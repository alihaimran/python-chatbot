# 🤖 Python Chatbot

A Python-based chatbot application that handles user queries and delivers automated, context-aware responses via the terminal.

## 🚀 Features
- Interactive terminal-based conversation
- Automated response system
- Lightweight and easy to use

## 🛠️ Technologies Used
- Python 3.x

## ▶️ How to Run
```bash
python main.py
```

## 📁 Project Structure
- `main.py` – Main chatbot logic
- `index.html` – Frontend interface
